# 求职打招呼语 / 私信专项

## 关于这个模块

国内求职的主动触达主要通过三个渠道：
- **BOSS 直聘**：求职者主动发出"打招呼"，HR 决定是否回复
- **脉脉**：职场社交+招聘，私信需要有人脉感或精准切入
- **领英（LinkedIn）**：外企/跨国公司场景，有 InMail 和 Connection Request

核心原则：**让对方在 5 秒内知道你是谁、你能做什么、为什么联系他。**

---

## 处理流程

### Step 1：确认用户信息

开始前确认（用户已提供的跳过）：
1. **目标岗位**：投的什么职位
2. **目标公司**（可选）：哪家公司的 HR 或内推人
3. **自己的背景**：最相关的 1-2 个亮点（学校/公司/年限/代表项目）
4. **渠道**：BOSS / 脉脉 / 领英 / 其他
5. **对方是谁**：HR / 内推人 / 用人经理（不同人写法不同）

### Step 2：生成打招呼语

根据渠道和对象输出对应版本。

## BOSS 直聘打招呼语

**字数建议**：50-80 字（超过 100 字 HR 几乎不看完）
**目的**：让 HR 想点开你的简历

**基本公式**：
```
[1 句话身份定位] + [1 句话与岗位最相关的亮点] + [明确行动请求]
```

**示例（前端开发，应届生）**：
> 「您好，我是 24 届应届生，React/TypeScript 技术栈，有 2 个上线项目（其中一个日活 5000+）。
> 对贵司 XX 产品很感兴趣，期待有机会聊聊。已投简历，麻烦您看一下，谢谢！」

**示例（运营，2 年经验）**：
> 「您好，我有 2 年抖音/小红书账号运营经验，主导过账号从 0 到 15 万粉的冷启动，
> ROI 稳定在 1:3 以上。看到贵司 XX 岗位很感兴趣，已投简历，期待连接。」

**示例（产品经理，转行）**：
> 「您好，我有 3 年互联网运营经验，今年系统转产品，已完成 3 份 PRD 和 Figma 原型。
> 对贵司 0 到 1 的产品阶段很感兴趣，期待了解这个岗位，已发送简历。」

**写法禁忌**：
- ❌ 「您好，我想应聘贵公司的 XX 职位，本人有 XX 经验，希望有机会」（官方废话，毫无亮点）
- ❌ 超过 100 字（HR 秒过）
- ❌ 复制粘贴模板感太强（在"贵司"这两个字上显露）
- ❌ 不写亮点，只写"已投简历请查收"（等于零信息量）

## 脉脉私信

**字数建议**：80-150 字（脉脉用户比 BOSS 更有社交感，稍长可接受）
**特点**：脉脉有真实人脉背景，对方可以看到你的职业信息，写法要更有"人情味"
**适用场景**：找内推 / 联系用人经理 / 社交转求职

**三段式结构**：
```
[为什么找这个人（共同背景/业内认同）] + [我是谁/最强亮点] + [希望对方做什么（具体请求）]
```

**示例（找内推）**：
> 「您好，看到您在 XX 公司做后端架构，我也在 Java 后端方向深耕了 3 年。
> 在我的上家公司主导过千万级 QPS 服务的稳定性优化，对分布式架构比较熟悉。
> 贵司最近有开 XX 方向的 HC，想请问是否方便帮忙内推或者给些建议？非常感谢！」

**示例（找用人经理）**：
> 「您好，我是一名有 5 年数据分析经验的从业者，看到贵司在搭建数据驱动的运营体系，
> 这个方向和我一直在做的事情非常匹配。
> 我主导过用户 RFM 分层和留存专项，对您最近分享的观点很有共鸣。
> 如果贵团队有数据分析方向的机会，非常期待有机会聊聊，打扰了！」

**写法要点**：
- 提及对方让你觉得"有缘由联系他/她"的内容（共同背景/他的文章/业界声誉）
- 请求要具体（"帮内推""给简历反馈""看看是否有合适岗位"），不要模糊
- 结尾要有礼节感，国内职场不需要西方那种强硬 follow-up 语气

## 领英（LinkedIn）私信 / InMail

**字数建议**：100-150 字（英文场景更精简）
**适用场景**：投外企 / 跨国公司 / 猎头联系

**英文版（简短型）**：
```
Hi [Name],

I came across your profile while researching [Company]. I'm a [role] with [X years] of experience 
in [most relevant skill/area], having [one specific achievement, e.g., "reduced API latency by 60% 
at ABC Corp"].

I noticed you're hiring for [position]. I'd love to learn more about the role. 
Would you be open to a quick chat?

Best,
[Your Name]
```

**中文版（如对方用中文）**：
> 「您好，在领英上看到您在 XX 公司负责 XX 方向的招聘。
> 我有 X 年 XX 经验，曾在 XX 公司主导过 XX（一句话成果）。
> 对贵司这个岗位很感兴趣，是否方便简短聊一下？谢谢！」

**领英禁忌**：
- ❌ 太正式（"谨此申请贵贵公司之职位"）
- ❌ 只发连接申请不附带消息（外企 HR 可能直接忽略）
- ❌ 立刻谈薪资

## 内推话术（找朋友/校友内推）

**前提**：认识但不是很熟的情况下，内推请求要给对方台阶下。

**示例（校友内推）**：
> 「学长/姐好，我是 [学校/年级]，在领英上看到您在 XX 公司。
> 最近在看 XX 方向的机会，贵司的 XX 岗位和我的经历比较匹配（[一句话亮点]）。
> 不知道是否方便帮我内推一下？如果不方便也完全理解，直接说就好。
> 这是我的简历链接/附件，麻烦您看看是否合适，谢谢！」

**要点**：
- 给对方留退路（"如果不方便也完全理解"）
- 简历/作品集放在消息里，降低对方的帮助成本
- 不要要求对方帮你"说好话"，只请求内推本身

## 各渠道长度参考速查

| 渠道 | 对象 | 建议字数 | 重点 |
|-----|------|---------|------|
| BOSS 直聘 | HR | 50-80 字 | 亮点 + 意向 |
| BOSS 直聘 | 用人经理 | 60-100 字 | 具体技能匹配 + 项目经历 |
| 脉脉私信 | 内推人 | 100-150 字 | 有缘由 + 请求具体 |
| 脉脉私信 | 用人经理 | 100-150 字 | 共鸣点 + 技能匹配 |
| LinkedIn（中文） | HR/招聘方 | 80-120 字 | 简洁 + 1 个亮点 |
| LinkedIn（英文） | HR/招聘方 | 60-100 words | Achievement + Ask |
| 校友内推 | 熟人 | 100-150 字 | 给台阶 + 简历已备 |

---

## 使用方式

1. 告诉我：目标岗位 + 你的最相关 1-2 个亮点 + 投递渠道
2. 可选：目标公司名、对方是 HR 还是用人经理、自己背景简述
3. 我输出 2-3 个版本供选择，你再根据实际情况微调
